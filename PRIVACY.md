# Privacy Policy — GitHub Easy Download

Last updated: July 2026

**GitHub Easy Download does not collect, store, transmit, or sell any user
data.**

## What the extension does

- Reads the current tab's URL (only on `github.com`) to identify which
  repository you're viewing.
- Makes a request to GitHub's public API (`api.github.com`) to fetch that
  repository's latest release information. This request is sent directly
  from your browser to GitHub — it does not pass through any server we
  operate, because we don't operate one.
- Reads your browser's `navigator.userAgent` locally, in your browser, to
  guess your operating system. This never leaves your device.

## What the extension does NOT do

- No analytics, tracking, or telemetry of any kind.
- No data is sent to the extension developer or any third party other than
  GitHub itself (which you're already visiting).
- No cookies, browsing history, or personal information is read or stored.
- No account login or authentication is required or requested.

## Third-party services

The only external service this extension talks to is GitHub's public REST
API (`api.github.com`), used solely to look up release files for the repo
you're currently viewing. See GitHub's own privacy policy for how they
handle that request: https://docs.github.com/en/site-policy/privacy-policies

## Contact

If you have questions about this policy, open an issue on the project's
GitHub repository.
